SetDrawYGrid

SetDrawYGrid — Whether or not to draw the Y grid lines

Synopsis

$plot->SetDrawYGrid($dyg)

Description

SetDrawYGrid enables or disables the drawing of the Y grid lines. (The Y grid lines are the horizontal lines which intersect the Y axis and are parallel to the X axis.) The default is to draw the Y grid.

Parameters

$dyg

True to draw the Y grid lines, False to not draw the Y grid lines.

Notes

SetLightGridColor sets the color of the grid lines.
