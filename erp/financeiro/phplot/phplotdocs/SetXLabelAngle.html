SetXLabelAngle

SetXLabelAngle — Set the text angle for X tick and data labels

Synopsis

$plot->SetXLabelAngle($xla)

Description

SetXLabelAngle sets the text angle for X tick and data labels. If using TrueType fonts, any angle can be used. If using built-in GD fonts, only 0 degree and 90 degree text can be used.

Parameters

$xla

Desired angle for label text, in degrees.

Notes

The default text angle for X labels is 0 degrees, for horizontal text.
